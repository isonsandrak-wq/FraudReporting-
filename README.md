This query is designed for a weekly fraud loss dashboard. It aggregates closed fraud claims by week and fraud type, calculating total loss per type, overall weekly loss, the percentage contribution of each type, and a benchmark average loss per type for that week. It enables fraud analysts to quickly see, for any selected week, which fraud types are driving losses above or below the weekly average — helping identify emerging fraud trends, allocate resources, and measure the impact of mitigation strategies.
